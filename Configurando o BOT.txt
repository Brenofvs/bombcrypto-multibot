Como configurar o BOT

1 - Baixe o zip code pelo github
https://github.com/rzanca/bombcrypto-multibot

2 - Descompacte o zip, de preferência na pasta C:/bomb

3 - Abra o arquivo config.yaml e modifique as informações se necessário

4 - Instale o Python 3.9.9 através do link abaixo, é necessário marcar a opção add Python to PATH!
https://www.python.org/downloads/release/python-399/

5 - Execute o arquivo "Instalar BOT.bat"

6 - Abra o BombCrypto, logue na metamask e abra o mapa.

7 - Deixe todas as janelas do BombCrypto em tela cheia.

8 - Execute o arquivo "Iniciar BOT.bat" e seja feliz :)
